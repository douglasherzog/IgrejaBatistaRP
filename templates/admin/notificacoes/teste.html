🔧 Configuração da API

Para usar em produção:

  1. Cadastre-se em uma API de WhatsApp (Twilio, Z-API, etc.)
  2. Obtenha suas credenciais (API Key, Instance ID, Token)
  3. Atualize as configurações em app/whatsapp_service.py
  4. Configure o webhook para receber respostas
  5. Teste com números reais

Nota: Atualmente o sistema está em modo de simulação para testes.
